Discord Message Wiper

Delete your own messages from a specific Discord server or channel. Everything runs in your browser — your token is never sent to anyone except Discord.

Read this first

  • Automating actions with a user account token is against Discord's Terms of Service (the “self-bot” rule), even when you're only deleting your own data. Use at your own risk — accounts can be flagged or disabled.
  • This tool can only delete messages you wrote. It cannot delete other people's messages.
  • Deletions are permanent and can't be undone. Use Dry run first.
  • Never share your token. Don't paste it into untrusted sites. The code for this page is open — inspect it before using it.

1. Credentials

How to find your token: open Discord in your browser → F12 (DevTools) → Network tab → reload → click any request to discord.com/api → copy the authorization request header. Don't enable “Disable cache” or refresh while DevTools is closed; Discord scrubs the value.

2. Where to wipe

Enable Developer Mode in Discord (Settings → Advanced), then right-click the server icon → Copy Server ID.

Right-click a channel → Copy Channel ID. Required for DM-style wipes; for guilds it scopes the search.
